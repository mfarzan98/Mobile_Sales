# Mobile_Sales
Mobile Sales Dashboard – Power BI
Short Description:
An interactive Power BI dashboard project designed to analyze mobile phone sales performance across different brands, models, regions, and time periods. The dashboard provides meaningful business insights using data visualization techniques to help understand sales trends, customer preferences, revenue generation, and market performance.

Technologies Used:
Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV
Data Cleaning & Transformation
Data Source
Dataset collected from Kaggle
Features / Highlights
Interactive and user-friendly dashboard design
Brand-wise and model-wise sales analysis
Revenue and profit insights visualization
Monthly and yearly sales trend analysis
KPI cards for total sales, revenue, quantity sold, and average sales
Dynamic filters and slicers for better analysis
Region/category-based performance comparison
Clean and visually appealing charts for quick decision-making
Data transformation and preprocessing using Power Query
Business insights generated using DAX measures
